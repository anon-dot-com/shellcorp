## Files

- `video_prompts.md` - All generation prompts for Replicate API
- `generate_video.sh` - Automated script to generate video via Replicate
- `README.md` - This file

## Usage

### Option 1: Automated Script
```bash
export REPLICATE_API_TOKEN="your_api_key"
chmod +x generate_video.sh
./generate_video.sh
```

### Option 2: Manual Generation
Use the prompts in `video_prompts.md` with:
- **luma/ray** for video clips (Luma AI Dream Machine)
- **prunaai/p-image** for still frames

### Combining Clips
```bash
ffmpeg -f concat -safe 0 -i clips.txt -c copy shellcorp_launch.mp4
```
